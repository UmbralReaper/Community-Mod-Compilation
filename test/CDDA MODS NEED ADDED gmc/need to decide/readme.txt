○ATM_and_Furnitures（家具と建築物(非公式MOD)）

	[追加 - 家具と建築物(非公式MOD)]
	(MOD識別名：ATM and Furnitures)

	ATMとその他の作れない家具を建設出来るようにするMODです。
	拠点を作ったはいいが、ATMが遠すぎる...という＠さんの悩みを解決するために作りました。
	
	<建設可能な家具リスト>
	・ATM
	・リサイクルマシン
	・CVD装置、CVD操作パネル
	・シャッターと巻き上げ機
	・その他の家具：洗面台、オーブン、トイレ、浴槽、洗濯機、乾燥機、シャワー、ビリヤード台、ゴミ箱、墓石、墓碑
	
	ATM、リサイクルマシン、CVD装置は先に製作で本体を作る必要があります。
	効果のある家具なので、そこそこの材料とスキルが必要になるようになっています。
	その他の家具は、今は別に効果はありません。飾りつけにどうぞ。

	CVDマシンは操作パネルが本体であり、CVD装置は実は必要ありません。飾りです。
	CVD装置のあるラボが遠い、寒すぎる、そもそも見つからない、といった＠さんのための追加です。
	グスタフ氏の研究ノートがないと作れないので、研究所へは行く必要があるでしょう。
	しかし、水素...

	シャッターと巻き上げ機の詳細については当Readmeの補足の項を参照。

	2016/06/01 リサイクルマシン、シンク、オーブン、トイレ、浴槽、ビリヤード台、ゴミ箱を追加
	2016/06/04 CVDマシンを追加

	2016/07/14 墓石と墓碑を追加
	2016/08/03 #5321に対応

○Uncraft_Reloaded_Ammo (手詰め弾薬分解MOD)

	[追加 - 手詰弾の分解(非公式MOD)]
	(MOD識別名：Uncraft_Reloaded_Ammo)

	手詰め弾薬はバニラでは分解することができません。

	自分で込めた弾薬がバラせないのは納得いかないですし、
	拾える手詰め弾は数が揃わないので使う機会に乏しく、
	なんとなく集めているうちに、邪魔になってこないでしょうか？
	そんな悩みに答えて、全種類の手詰め弾を分解可能にします。
	ただし分解で得られる火薬は作る際の半分まで、となっています。
	弾頭、薬莢、ジャケット、散弾などは製作時と同じ数だけ回収できます。

	2016/07/01 #5144で削除された40mmの分解レシピを削除
	2016/09/16 .22CB、.22ラットショット、12ゲージビーンバッグ弾の削除に対応
	2016/09/17 #5489での分解レシピの記述方法変更に対応


○Useful_Goods（便利グッズMOD）

	[追加 - 便利グッズ(非公式MOD)]
	(MOD識別名：Useful_Goods)

	あれがあったらいいのになー、という個人的な願いを具現化したMODです。

	<追加アイテムリスト>
	・携帯スキミング装置	:ATMの全振替を行える携帯ツール。これでインベントリがスッキリ！手持ちのカードのうち、もっとも高額なカードに振替えます。
	・ガレージジャッキ		:20tの重量でも持ち上がる油圧式ジャッキ。アームで吊ってタイヤ交換とかありえないです。類似品と同じ場所で登場します。製作することも可能です。
	・高耐久腕時計			:ずばりあのブランドの時計。温度、湿度、気圧、季節と日の出/日の入り時刻を表示。ダイバーズウォッチとほぼ同じ場所に落ちてます。
	・標的(射撃練習用)		:設置できる的。射撃練習に使える。(ブロブ扱いなのでモンスターからは無視されるはず)
	・ダーツボード			:同上

	※製作可能なもののレシピはその辺の関連してそうな本で覚えます。

	<その他の変更>
	・パンティを[a](使う)と頭に装備出来るようになります。

	＞補足
	携帯スキミング装置は同額の複数のキャッシュカードがあった場合、そのうち1枚しか振り替えてくれません。
	(使用1回で1枚は振り替えられるので繰り返し使用すればOK)

○Besieged_Survivor（包囲された生存者MOD）

	[追加 - 生存者(包囲)(非公式MOD)]
	(MOD識別名：Besieged Survivor)

	いくつかの固定NPCと、彼らの立て籠もるマップを追加するMODです。
	NPCはいずれも高度なスキルを有しており、救出して信頼関係を築ければ、大いに助けになってくれるでしょう。
	ただし、モンスターに包囲されており、救出するには相応の実力が必要になります。
	追加マップには下記NPCが居ない場合もあります。

	※派閥が追加されるため、ワールドのリセットか再作成が必要です。
	※公式の「高層ビル」MOD、および非公式の「便利グッズMOD」が必須です。
	※固定NPCをオフにしているとマップのバリエーションが増えるだけになります。

	<追加NPC>
	・整備士		：機械整備スキル最大20 / 自動車修理工場に立て籠もっている
	・兵士			：サバイバル、射撃、各銃器スキル最大8 / 掩体壕で兵士ゾンビに包囲されている
	・弁護士		：話術、交渉スキル最大15 / ゾンビに取り囲まれ、オフィスのトイレから出れなくなっている
	・銃整備士		：機械整備スキル最大10、射撃、各銃器スキル最大8 / ガンショップに立て籠もっている
	・警察官		：近接戦闘最大6、射撃、ハンドガンスキル最大6 / 警察署に立て籠もっている
	・バーテンダー	：話術スキル最大15、ランチャースキル最大8 / 酒場でゾンビ相手に商売している
	・運転手		：運転スキル最大15、近接戦闘スキル最大8 / ガソリンスタンドでタンクローリーを守っている
	・おまえら		：コンピュータースキル最大15 / 自宅警備員だお

○Save_the_Milirary_Outpost (前哨地を守れ！MOD)

○シャッターと巻き上げ機について

	巻き上げ機を操作すると、条件に一致するタイルがシャッター(閉じた金属ドア)←→床 と変化します。

	慣れるまでは、建設前にセーブ推奨です。設置場所を間違うと、
	シャッターが出ない、意外な場所にシャッターが出現、シャッターで圧死、といった事が起きます。
	なお既存のロックされたドアに対して設置すると、そのドアが開いちゃいますｗ


	仕様：
	・巻き上げ機が「壁」に接していること
	・上記「壁」の直線方向に「壁」がある
	・上記「壁」同士が「床」でつながっている
	・「壁」と「床」が巻き上げ機に対応している
	・上記すべてを満たすと、「壁」と「壁」の間がシャッター化する

	対応している壁
	・壁(家屋の普通のやつ) (色付きの壁も当MODで追加したのでOK)
	・壁(コンクリート)
	・壁(レンガ)
	・壁(金属)

	対応している床
	・床(家屋の普通のやつ)

	例)
	６...巻き上げ機
	■...壁
	木...壁(木)
	／...シャッター
	ｗ...草地
	
	                                 ↓(対応していない壁なので出ない)
	|------------------|      |------------------|
	|      木          |      |      木          |
	|                  |      |                  |
	|                  |      |                  |
	|■ｗｗ■        ■|  →  |■ｗｗ■／／／／■|←(シャッターが出現)
	|      ６          |      |      ６          |
	|      ＠          |      |↑    ＠          |
	|------------------|      |↑----------------|
	                           ↑(「床」でつながっていないので出ない)
