How to install:

1. Find your "cdda" folder

2. Open the cdda folder and then open the "data" folder

3. Open the "mods" folder

4. Put the Playtime folder in the mods folder

5. To activate the mod in-game you must create a new world 

6. Find "Playtime" in the mod list under "Default" and select it

7. The items in the mod should now spawn in the world and be spawnable through the debug menu
