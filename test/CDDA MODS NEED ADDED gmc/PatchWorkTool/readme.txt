
CDDA_PWT Ver_0.10

■概要

Cataclysm dda 用のタイル追加支援用ツールです。

画像と単純な構造のデータを貼り付けるのが主な機能となります。
背景やエフェクト、車両パーツ等の複雑なデータは基本的に処理対象外です。


・MShock32系のタイルを基準にして作成したツールなのでそれ以外では基本的に動作しないものとお考え下さい。
（※ChestHole系などで記述のある sprite の配列型には対応していません。
　その辺のデータは取り込まれませんので自前でカスタムしている人もご注意ください。）

	事例	〇読込可能　　{ "weight":50, "sprite":2050}
			×読込不可能　{ "weight":1, "sprite":[2918, 2913, 2918, 2919]},


・処理速度は申し訳ないけどかなり遅いです。（特にデータ読み込み処理）


・もしかしたらメモリリークとかかましてるかもしれませんので、
　使ってる内に「なんか処理が遅くなってきたような・・・？」と感じたら、
　お手数ですがＰＣを再起動してみて下さい。
